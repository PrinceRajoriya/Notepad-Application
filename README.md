# 📝 Python Notepad Application

A simple desktop text editor built using **Python** and **Tkinter**. The application provides a clean graphical interface for creating, editing, opening, and saving text files using Python's File I/O operations.

---

## 🚀 Features

- Create new text files
- Open existing text files
- Edit text
- Save files
- Save As functionality
- User-friendly GUI
- Exception handling for file operations

---

## 🛠 Tech Stack

- Python
- Tkinter
- File I/O
